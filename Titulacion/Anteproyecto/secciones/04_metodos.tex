\section{Materiales y métodos}

\hspace{1cm}\textbf{Extracción de las secuencias en del grupo K01220 en KEGG}
	
Se utilizara la API de KEGG \parencite{Kanehisa_2004, Kanehisa_2022} para la extracción automática de los datos. La documentación oficial de la misma está disponible en \url{https://www.kegg.jp/kegg/rest/}. Se escribirán diversos scripts de python (versión 3.12.7) en cuadernillos interactivos de python (archivos .ipynb) con el obejtivo primordial de:
\begin{itemize}
	\item Extraer las secuencias del grupo K01220.
	\item Limpieza de datos.
	\item Generación de archivos. 
\end{itemize}
