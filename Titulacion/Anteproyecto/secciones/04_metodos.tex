\hspace{1cm}\textbf{Extracción de las secuencias}
	
Se escribirá un script en python (versión 3.14.xx) para la obtención de las secuencias del grupo K01220. Será un script que consuma la API de KEGG \parencite{Kanehisa_2004, Kanehisa_2022}. La documentación oficial está disponible en \url{https://www.kegg.jp/kegg/rest/}
